Diagram Style & Format

Pick the format that fits how you'll use these posters.

A

HTML + SVG Poster Pages

One self-contained HTML file per diagram. Rich colors, clean lines, printable. Opens in any browser, no tools needed. Best for sharing, presenting, or printing as A3/A4 posters. Recommended.

Pros

  • No install required
  • Looks great on screen & print
  • Fully portable

Cons

  • Not editable in diagram tools
B

Mermaid Markdown (.md)

Text-based diagrams embedded in Markdown. Renders natively on GitHub, Notion, Obsidian, GitBook. Easy to version-control and edit by hand.

Pros

  • GitHub renders inline
  • Easy to edit/diff

Cons

  • Limited visual polish
  • Complex diagrams can be awkward
C

Both — HTML posters + Mermaid source

Each diagram gets an HTML poster for display and a Mermaid .md for documentation and version control. Best of both worlds.

Pros

  • Great visuals AND editable source
  • Works in docs and presentations

Cons

  • More files
